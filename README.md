Telegram Reminder Bot

A Telegram reminder bot with a dashboard interface for scheduling reminders automatically.

Features

- Set reminders using dashboard
- Automatic Telegram notifications
- Backend reminder scheduler
- Simple user interface

Technologies Used

- Node.js
- JavaScript
- Telegram Bot API
- HTML
- CSS

Future Improvements

- User authentication
- Database integration
- Cloud deployment
- Real-time notifications
